# Structure du Site BFC Belgium

## Architecture des Pages

```
www.bfcbelgium.com/
├── / (Accueil)
├── /products/
│   ├── /burgers
│   ├── /buckets
│   ├── /desserts
│   └── /drinks
├── /deals
│   ├── Meals
│   └── Menus
├── /restaurants
├── /privacy-policy
└── /terms-of-use
```

---

## Navigation Principale (Header)

| Élément | Lien | Description |
|---------|------|-------------|
| Logo BFC | `/` | Retour à l'accueil |
| Products | `/products/burgers` | Menu déroulant vers catégories |
| Deals | `/deals` | Promotions et menus |
| Restaurants | `/restaurants` | Liste des restaurants |
| EN / FR / NL | - | Sélecteur de langue |

- Titre: "Explore Products"

  - soHot (ailes épicées)
  - Crunchy (poulet croustillant)
  - Menu (formules)
- Bouton CTA: "Explore Products" → `/products/burgers`

### Section Restaurants
- Titre: "Our Restaurants"
- Bouton CTA: "Our Restaurants" → `/restaurants`

### Section Galerie
- Images d'ambiance du restaurant
- Photos de produits
